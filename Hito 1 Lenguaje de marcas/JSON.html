JSON o (JavaScript Object Notation)

Historia de JSON

JSON es un formato de texto sencillo para el intercambio de datos formado por subconjuntos de objetos de JavaScript.Debido a su amplio uso como alternativa a XMl, JSON se considera independiente de XML. JSON es usado habitualmente en entornos donde el volumen de datos entre cliente y servidor es de vital importancia, por eso se usa en navegadores como Yahoo,Google,Mozilla etc....

Si bien JSON es una alternativa de XML no es raro ver el uso de los dos en aplicaciones.

Sintaxis de JSON

La sintaxis de JSON se basa en estos datos disponibles

Numeros:Se permiten numeros negativos y tambien partes separadas por puntos. (-678.912)

Cadenas:Estos representan secuencias de cero o mas caracteres,se ponen dentro de comillas. ("Soy listo")

Booleanos:Representan valores boolenos pueden ser ("true") o ("false")

Null:Representan valor nulo

Array:Representa una lista ordenada de cero o mas valores los valroes se separan por comas y el vector se mete en corchetes.["alejandro","daniel","samuel"]
